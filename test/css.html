Overview

The goal of this file is to test out the iconpark.css file distributed with the gem to make sure it's a reasonable starting place. To view run rake test.

The icons provided by iconpark supports up to 4 different "themes" but not all icons make use of these themes. Therefore I'm going to show a grid of an example icon that is naturally one of each theme on each column, but then the row is the applied theme. Each intersection of that grid will show the icon on both with a dark background and a light background.

The icons are purposely over-sized to make the detail more obvious. Also this file is not actually using the asset pipeline but instead a special test server and plain HTML for simplicity.
